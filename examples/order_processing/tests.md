# 测试用例设计

## 程序
- 程序名: ORDPRC

## 测试用例列表

### TC-001 正常处理 - 状态 02
- 前置条件: ORDPF 中存在状态=01 且客户/库存有效的订单
- 步骤:
  1. 执行程序
  2. 查询 ORDPF 状态
- 预期结果:
  - 订单状态更新为 02
  - ORDLOGPF 记录“订单确认完成”

### TC-002 库存不足 - 状态 09
